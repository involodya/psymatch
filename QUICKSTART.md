# Быстрый старт

## Шаг 1: Создайте Telegram бота

1. Откройте Telegram и найдите @BotFather
2. Отправьте команду `/newbot`
3. Следуйте инструкциям и получите токен бота
4. Сохраните токен - он понадобится для настройки

## Шаг 2: Узнайте свой Telegram ID

1. Найдите бота @userinfobot в Telegram
2. Отправьте ему любое сообщение
3. Скопируйте ваш ID (это число)

## Шаг 3: Установите зависимости

```bash
pip install -r requirements.txt
```

## Шаг 4: Настройте бота

### Вариант 1: Автоматическая настройка
```bash
python3 setup.py
```

### Вариант 2: Ручная настройка
1. Скопируйте `env.example` в `.env`:
```bash
cp env.example .env
```

2. Откройте `.env` и заполните:
```
BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_IDS=ваш_telegram_id
DATABASE_PATH=psymatch.db
LOG_FILE=bot.log
```

## Шаг 5: Запустите бота

```bash
python3 bot.py
```

## Шаг 6: Начните использовать

1. Найдите вашего бота в Telegram по username
2. Отправьте `/start`
3. Выберите роль (психолог или пациент)
4. Заполните профиль
5. Пройдите психологический тест

## Кастомизация

### Изменить вопросы теста

Отредактируйте файл `test_questions.json`. Каждый вопрос должен содержать:
- `question` - текст вопроса
- `options` - 5 вариантов ответа
- `weights` - 5 весов (от -1 до 1) для расчета совместимости

### Изменить тексты сообщений

Отредактируйте файл `messages.json` - там находятся все текстовые сообщения бота.

## Добавление администраторов

Чтобы добавить администратора (для доступа к статистике):
1. Узнайте его Telegram ID через @userinfobot
2. Добавьте ID в `.env` файл в параметр `ADMIN_IDS` через запятую:
```
ADMIN_IDS=123456789,987654321,111222333
```
3. Перезапустите бота

## Troubleshooting

### Бот не отвечает
- Проверьте правильность токена в `.env`
- Убедитесь, что бот запущен (`python3 bot.py`)
- Проверьте логи в файле `bot.log`

### Не работает база данных
- Убедитесь, что у вас есть права на запись в папке проекта
- Проверьте путь к базе данных в `.env`

### Проблемы с тестом
- Убедитесь, что `test_questions.json` содержит корректный JSON
- В каждом вопросе должно быть ровно 5 вариантов ответа и 5 весов
